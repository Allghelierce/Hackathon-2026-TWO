# Titan Addresses Dataset

This CSV file (`titan-addresses.csv`) contains permit and address information for various businesses, with a focus on solar power contractors and related companies.

## Data Structure

The dataset includes the following columns:

- **BIZ_NAME**: Business name or company name
- **NAME**: Contact person or representative name
- **PRIMARY_EMAIL**: Primary email address for contact
- **PRIMARY_PHONE**: Primary phone number
- **LICENSE**: Business license number
- **PERMIT_ID**: Unique permit identifier (appears to be a hashed/encoded ID)
- **STREET_NO**: Street number of the address
- **STREET**: Street name
- **CITY**: City name
- **STATE**: State abbreviation
- **ZIPCODE**: ZIP code
- **PERMIT_DATE**: Date when the permit was issued (YYYY-MM-DD format)

## Data Overview

- Total records: Approximately 91,000 entries
- Primary focus: Solar power contractors, particularly Titan Solar Power across multiple states
- Geographic coverage: Multiple US states including CA, NM, AZ, FL, IL, CO, etc.
- Date range: Permits from various years, with recent entries up to 2023

## Usage

This dataset can be used for:
- Analyzing solar permit trends
- Mapping contractor locations
- Contact information for solar businesses
- Geographic analysis of solar installations

## Source

This data appears to be compiled from public permit records and business registrations for solar energy contractors and related businesses.
